About the Project

SkillConnect is a web application developed using React.js that allows users to discover, compare, and explore online courses across multiple platforms. It was created to simplify the search for suitable online learning opportunities based on usersâ€™ skill levels, interests, and goals.

The project was completed as part of the internship requirement under the guidance of Dr. P. Srilatha, Associate Professor, Department of AI & DS, Chaitanya Bharathi Institute of Technology (CBIT), Hyderabad.

Project Objectives

  • Develop a responsive front-end web application using React.js.
  • Implement keyword-based search and filtering for courses.
  • Integrate real-time course data dynamically via APIs.
  • Apply modular, reusable component architecture for scalability.
  • Enhance user experience through clean UI and intuitive navigation.

Core Features

  • Search courses by keywords or topics.
  • Filter by category, difficulty, and duration.
  • Responsive layout compatible across devices.
  • Dynamic API data fetching with optimized loading.
  • Minimalist and intuitive interface design.
